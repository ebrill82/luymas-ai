# Luymas AI - Desktop Application

Built with **Tauri + React**

## Getting Started

### Prerequisites
- [Rust](https://rustup.rs/)
- [Node.js](https://nodejs.org/)

### Install and Run

```bash
npm install
npm run tauri dev
```

### Build for Production

```bash
npm run tauri build
```

## Tech Stack

- **Framework**: Tauri 2.0
- **Frontend**: React + TypeScript
- **Backend**: Rust
- **Platforms**: Windows, macOS, Linux

## Features

- Native desktop performance
- Small binary size (< 10MB)
- Cross-platform (Windows, macOS, Linux)
- Shared business logic with web and mobile versions
- Luymas API key integration for monitoring
