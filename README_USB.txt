  1. Branchez la clé USB sur votre ordinateur
  2. Double-cliquez sur "start.bat" (à la racine de la clé)
  3. Le Studio s'ouvre automatiquement dans votre navigateur
     à l'adresse : http://localhost:5000
  4. Commencez à utiliser Luymas AI !

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  • Ne retirez PAS la clé USB pendant l'utilisation
  • Ollama sera installé automatiquement si absent
  • Les modèles IA seront téléchargés selon votre matériel
  • Pour arrêter : appuyez sur Ctrl+C dans la fenêtre de commande
  • Attendez l'arrêt complet avant de retirer la clé

  💻 CONFIGURATION RECOMMANDÉE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Minimum :
    • Windows 10/11 (64 bits)
    • 8 Go de RAM
    • 10 Go d'espace libre sur la clé
    • Connexion Internet (pour le premier lancement)

  Recommandé :
    • 16 Go de RAM ou plus
    • GPU NVIDIA avec 8 Go VRAM ou plus
    • 30 Go d'espace libre
    • Connexion Internet stable

  🐛 DÉPANNAGE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Problème : "Ollama non trouvé"
  Solution : Le script télécharge Ollama automatiquement.
             Si le téléchargement échoue, installez manuellement :
             https://ollama.com/download/windows

  Problème : "Python non trouvé"
  Solution : Installez Python 3.10+ :
             https://www.python.org/downloads/
             ⚠️ Cochez "Add Python to PATH" lors de l'installation

  Problème : Le Studio ne s'ouvre pas
  Solution : Ouvrez manuellement : http://localhost:5000

  Problème : Modèles trop lents
  Solution : Utilisez des modèles plus petits ou ajoutez un GPU.
             Modifiez le tier dans le fichier .env
